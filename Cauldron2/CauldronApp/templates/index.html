All-in-one

Cauldron is a SaaS analytics solution that allows users to aggregate information from multiple data repositories of different types of data sources that define their custom projects. Because software development is more than just code commits in a single repository.

Shareable Reports

It's been said that "collecting data is only the first step toward wisdom, but sharing data is the first step toward community"(Henry Louis Gates, Jr.). Cauldron allow users to share reports and metrics about software development.

100% Free, Open Source

Cauldron is, and will always be, Free as in Freedom to allow users, contributors, and maintainers to use, to study, to share, and to improve its software openly.
